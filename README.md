# NodeJS API - TypeScript

## Installation
#### Set up
First of all, install all dependencies..
```bash
$ npm|yarn install
```
#### Development
To run it in `watch` mode using Nodemon, run the following command..
```bash
$ npm|yarn run watch
```
#### Tests
To run the tests for this project, run the following command..
```bash
$ npm|yarn test
```
#### Build
```bash
$ npm|run build:prod
```
